# **Wanderlust Korea - Travel & Culture Portal**

## **개요**
대한민국의 구석구석 숨겨진 여행지와 실시간 문화 행사 정보를 제공하는 **국내 여행 특화 큐레이션 플랫폼**입니다. 최신 여행 뉴스와 지역별 축제 정보를 한눈에 파악하고, 자신의 여행 기록을 공유할 수 있습니다.

## **상세 기능 및 구현 현황**

### **1. 통합 헤더 및 사용자 인증 (Header - Authentication)**
- **단일 뷰 아키텍처:** 별도의 로그인 페이지 없이 대시보드에서 즉시 정보를 탐색할 수 있는 구조.
- **실시간 로그인/로그아웃:** 헤더 우측에 닉네임 입력 및 로그인 기능을 통합. 로그인 시 사용자 닉네임과 [로그아웃] 버튼 표시.
- **인터랙션 제어:** 로그인이 필요한 기능(예: 추천 공유)은 로그인 여부를 확인하여 사용자에게 알림 제공.

### **2. 실시간 여행 뉴스 & 정보 (Top-Left)**
- **고품질 뉴스 큐레이션:** 국내 여행 기사(서울 벚꽃 1위, 군산/진주/청주 핫플레이스 등)를 엄선하여 제공.
- **실시간 업데이트 버튼:** 사용자가 직접 클릭하여 최신 뉴스를 불러올 수 있는 [↻] 버튼 추가.

### **3. 지역별 축제 & 전시 가이드 (Top-Right)**
- **하이브리드 실시간 달력:** 정적 2026년 주요 축제 데이터와 RSS를 통해 수집된 실시간 행사 정보를 결합하여 표시.
- **실시간 업데이트 버튼:** 달력 패널의 [↻] 버튼을 클릭하면 최신 뉴스 및 행사 정보를 즉시 새로고침.
- **스마트 날짜 파싱:** RSS 정보의 제목에서 날짜를 자동으로 추출하여 달력의 해당 날짜에 자동으로 배치.
- **콘텐츠 구분:** 정적 데이터는 🚩, 실시간 데이터는 🆕 아이콘과 함께 초록색 강조 점으로 구분 표시.

### **4. 여행자 추천 공간 (Bottom - Social Feed)**
- **장소 추천:** 사용자가 다녀온 여행지나 추천하고 싶은 장소를 닉네임과 함께 기록.
- **인기 여행 유튜브:** 대한민국 구석구석 등 고품질 여행 채널의 최신 영상 큐레이션.

## **Current Implementation Plan**
1.  **Integrated Header Login:** 첫 화면의 로그인 기능을 페이지 상단 헤더로 이동하여 사용자 경험 개선.
2.  **Hybrid Real-time Calendar:** 정적 축제 데이터 보강 및 RSS 연동을 통한 실시간 달력 업데이트 기능 구현.
3.  **UI Polish:** 실시간 데이터 강조를 위한 뱃지 및 색상 시스템 적용.
